# Agentic-Law-Firm (Claude Code entry)

This file is loaded automatically by Claude Code when you open this
project. It is intentionally short — all operating rules live in
`AGENTS.md` at the project root, which is shared across Claude Code,
Google Antigravity, and any other model-agnostic agent harness.

Read `AGENTS.md` for:
- Project mission and persona
- Routing model (single operator -> task-router -> specialists)
- Hard gates (high-risk actions that require human approval)
- Document style discipline (no emoji, no decorative colour, black on
  white, follow `docs/DOCUMENT_STYLE_GUIDE.md`)
- How to add or modify agents, skills, and MCP connectors

First-time setup: type `/onboard` and follow the wizard.
