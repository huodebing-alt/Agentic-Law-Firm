# Sg Copyright Assignment Drafting

## Inputs
- Works
- Parties
- Scope and consideration

## Workflow
1. Apply CA 2021 assignment formality (writing, signed by assignor)
2. Capture moral rights position
3. Draft assignment / licence
4. Output instrument

## Output
Copyright assignment deed.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely per jurisdiction convention.
