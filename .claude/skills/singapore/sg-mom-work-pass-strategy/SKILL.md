# Sg Mom Work Pass Strategy

## Inputs
- Role, salary, qualifications
- Employer track record

## Workflow
1. Apply EP salary floors (general / financial-services / age-tiered)
2. Apply COMPASS framework points
3. Check S Pass quota and levy
4. Run ONE Pass eligibility (S$30k base / sectoral exceptions)
5. Output recommendation memo

## Output
Work-pass strategy memo with application checklist.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely per jurisdiction convention.
