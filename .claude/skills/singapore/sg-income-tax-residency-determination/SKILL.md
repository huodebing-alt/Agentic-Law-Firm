# Sg Income Tax Residency Determination

## Inputs
- Travel calendar / management-and-control location

## Workflow
1. Apply individual quantitative test (183 days)
2. Apply company management-and-control test
3. Apply DTAA tie-breaker if dual residency
4. Output residency memo

## Output
Residency determination memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely per jurisdiction convention.
