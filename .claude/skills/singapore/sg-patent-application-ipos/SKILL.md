# Sg Patent Application Ipos

## Inputs
- Invention disclosure
- Priority application if any

## Workflow
1. Run prior-art search via sg-ipos and external databases
2. Draft claims and specification (PCT format)
3. File with IPOS
4. Calendar search and examination
5. Output application package

## Output
IPOS patent application packet.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely per jurisdiction convention.
