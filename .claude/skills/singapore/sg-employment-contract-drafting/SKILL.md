# Sg Employment Contract Drafting

## Inputs
- Employee category
- Role and pay
- Hours, leave, benefits
- Restrictive covenants

## Workflow
1. Choose template (Part IV employee, executive, fixed-term)
2. Apply EA minimums; flag any contracting-out
3. Insert PDPA / IP assignment / TAFEP-compliant clauses
4. Format per SG style (Times New Roman 12, 1.5 spacing, A4)
5. Output docx via templates/sg/employment-contract-sg.docx

## Output
Employment contract in SG style.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely per jurisdiction convention.
