# Sg Probate Application

## Inputs
- Deceased details
- Will / intestacy facts
- Assets schedule

## Workflow
1. Apply Probate & Administration Act
2. Prepare ex parte originating summons
3. Prepare administration oath and supporting affidavits
4. Output probate / LA application

## Output
Probate application packet.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely per jurisdiction convention.
