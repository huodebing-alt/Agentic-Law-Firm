# Sg Mas Compliance Check

## Inputs
- Entity type and licence
- Activity log

## Workflow
1. Pull applicable MAS Notices and Guidelines
2. Map AML/CFT, conduct, and reporting obligations
3. Identify deficiencies
4. Output compliance gap memo

## Output
MAS compliance gap memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely per jurisdiction convention.
