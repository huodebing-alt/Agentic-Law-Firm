# Sg Pdpa Impact Assessment

## Inputs
- Processing description
- Data subjects and categories
- Recipients and transfers

## Workflow
1. Map data flows
2. Identify consent / notification basis
3. Assess transfer adequacy (NICs, contractual clauses)
4. Rate harm likelihood / severity
5. Output DPIA memo and mitigation list

## Output
PDPA DPIA memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely per jurisdiction convention.
