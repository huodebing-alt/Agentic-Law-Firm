# Sg Trademark Application Ipos

## Inputs
- Mark
- Goods and services per Nice classification
- Owner

## Workflow
1. Run clearance via sg-ipos search
2. Classify goods/services per Nice
3. Apply absolute and relative grounds check
4. Lodge application and pay official fees
5. Calendar examination and opposition window

## Output
IPOS trademark application packet.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely per jurisdiction convention.
