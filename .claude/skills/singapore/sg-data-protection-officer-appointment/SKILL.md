# Sg Data Protection Officer Appointment

## Inputs
- Org chart
- Candidate DPO details

## Workflow
1. Draft appointment letter and job description
2. Map DPO reporting line and resources
3. Register DPO contact for public-facing publication
4. Draft DPO board-level reporting cadence
5. Output appointment package

## Output
DPO appointment letter plus internal policy.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely per jurisdiction convention.
