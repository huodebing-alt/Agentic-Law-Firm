# Sg Maintenance Application

## Inputs
- Applicant and respondent details
- Income / expense schedule

## Workflow
1. Apply WC ss. 69-71
2. Compute reasonable quantum based on means
3. Draft Form 90 application and supporting affidavit
4. Output maintenance application

## Output
Maintenance application packet.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely per jurisdiction convention.
