# Us Trademark Application Us

## Inputs
- Mark
- Goods/services
- Owner

## Workflow
1. Clearance via us-uspto
2. Classify per Nice and USPTO ID Manual
3. File TEAS Plus / Standard
4. Calendar examination
5. Output filing receipt and checklist

## Output
USPTO TM filing packet.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely per jurisdiction convention.
