# Us Patent Claim Drafting

## Inputs
- Disclosure
- Independent vs dependent claim count

## Workflow
1. Apply 35 USC 112 written description / enablement / definiteness
2. Draft independent + dependent claims
3. Output claim set

## Output
Claim set.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely per jurisdiction convention.
