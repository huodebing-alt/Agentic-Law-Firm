# Us Delaware Incorporation

## Inputs
- Entity type
- Capital
- Directors
- Registered agent

## Workflow
1. File Certificate of Incorporation / Formation with DE Secretary of State
2. Adopt bylaws / operating agreement
3. Hold organizational meeting and issue shares
4. Register for EIN and DE franchise tax
5. Output incorporation packet

## Output
DE incorporation packet.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely per jurisdiction convention.
