# Bundle Board Pack

## Inputs
- Agenda
- Board papers

## Workflow
1. Sequence papers per agenda
2. Build index and pagination
3. Add cover page
4. Output .pdf

## Output
Board pack .pdf.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
