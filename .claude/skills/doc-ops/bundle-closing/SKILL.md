# Bundle Closing

## Inputs
- Closing checklist
- Source files

## Workflow
1. Collect files per the closing checklist
2. Build index and pagination
3. Add signature pages
4. Output .pdf and ZIP

## Output

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
