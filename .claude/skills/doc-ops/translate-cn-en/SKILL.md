# Translate CN <-> EN

## Inputs
- Document
- Source / target language

## Workflow
1. Extract text
2. Apply terminology consistency
3. Preserve citations exactly
4. Output side-by-side or single-language version
5. Hand to requesting agent

## Output
Translated document.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
