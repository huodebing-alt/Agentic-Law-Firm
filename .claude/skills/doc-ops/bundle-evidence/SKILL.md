# Bundle Evidence

## Inputs
- Source files
- Bundle name

## Workflow
1. Collect files in order
2. Build the index
3. Paginate continuously
4. Add bookmarks
5. Output .pdf

## Output
Evidence bundle .pdf.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
