# Anonymise Document

## Inputs
- Document
- Names to anonymise

## Workflow
1. Identify names and substitute placeholders
2. Keep structural references consistent
3. Output anonymised .docx and a substitution log

## Output
Anonymised .docx + substitution log.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
