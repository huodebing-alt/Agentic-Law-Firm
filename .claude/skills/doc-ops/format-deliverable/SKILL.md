# Format Deliverable

## Inputs
- Source .docx or Markdown

## Workflow
1. Read templates/_style/style-config.json
2. Apply page setup, fonts, line spacing, paragraph rules
3. Output .docx

## Output
Finalised .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
