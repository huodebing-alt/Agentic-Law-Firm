# Side-by-Side CN/EN

## Inputs
- Source document
- Source language

## Workflow
1. Extract text
2. Translate via doc-ops-translator
3. Lay out side-by-side per style guide
4. Output .docx

## Output
Side-by-side .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
