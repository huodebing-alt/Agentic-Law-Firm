# PIPL Cross-Border Transfer

## Inputs
- Data categories
- Volumes
- Recipient jurisdiction
- Purpose

## Workflow
1. Apply CAC thresholds
2. Pick pathway
3. Draft PIIA
4. Draft standard contract or filing
5. Output package

## Output
Cross-border transfer package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
