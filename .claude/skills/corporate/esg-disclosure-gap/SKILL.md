# ESG Disclosure Gap

## Inputs
- Current disclosure
- Listing venue / regime

## Workflow
1. Pull regime requirements
2. Map current vs required
3. Identify gaps and prioritise
4. Output gap memo
5. Hand to esg-specialist

## Output
Gap memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
