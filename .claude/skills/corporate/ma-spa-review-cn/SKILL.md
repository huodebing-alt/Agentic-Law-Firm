# M&A SPA Review (PRC)

## Inputs
- SPA .docx
- Client side (buyer / seller)

## Workflow
1. Run checklist
2. Highlight one-sided clauses
3. Identify missing standard clauses
4. Cross-check governing-law enforceability
5. Output review memo

## Output
Review memo .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
