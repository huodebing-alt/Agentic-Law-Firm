# FIE Record Filing

## Inputs
- FIE name
- Change description

## Workflow
1. Confirm change requires record filing
2. Draft the filing form
3. Compile supporting docs
4. Submit checklist
5. Output package

## Output
MOFCOM record filing package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
