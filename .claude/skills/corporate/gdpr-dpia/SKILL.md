# GDPR DPIA

## Inputs
- Processing operation description
- Data subjects
- Categories
- Recipients

## Workflow
1. Apply Art. 35 criteria
2. Identify risks and mitigations
3. Draft DPIA
4. Coordinate with PIPL if dual track
5. Output DPIA

## Output
DPIA document.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
