# M&A DD Checklist (PRC)

## Inputs
- Target name
- Deal type (equity / asset)
- Sector
- Cross-border (yes / no)

## Workflow
1. Pull sector-specific items (data, IP, real estate, regulatory)
2. Pull general items (corporate, contracts, employment, tax, litigation)
3. Tag each item with priority (red / amber / green)
4. Output as table .docx
5. Hand to ma-specialist for tailoring

## Output
DD checklist .docx with 10 main sections, ~150 items.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
