# IPO Prospectus Section

## Inputs
- Listing venue
- Section requested
- Underlying facts

## Workflow
1. Pull the listing rule for the section
2. Draft per disclosure standard
3. Cross-check facts
4. Hand to hard-gate-keeper
5. Output draft

## Output
Prospectus section draft.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
