# National Security Review (PRC)

## Inputs
- Investor
- Target
- Sector
- Control level

## Workflow
1. Apply NSR triggers (sensitive sectors, control acquisition)
2. Draft the notification
3. Compile supporting docs
4. Coordinate with M&A
5. Output package

## Output
NSR notification package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
