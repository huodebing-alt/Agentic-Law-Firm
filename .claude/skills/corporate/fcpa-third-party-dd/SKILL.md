# ABAC Third-Party DD

## Inputs
- Counter-party name
- Country
- Activity

## Workflow
1. Pull public-record screen (tianyancha / qichacha / world-check)
2. Run ABAC questionnaire
3. Score the risk
4. Recommend mitigations
5. Output DD memo

## Output
ABAC DD memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
