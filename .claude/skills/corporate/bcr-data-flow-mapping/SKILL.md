# BCR / Data Flow Mapping

## Inputs
- Org chart
- Systems and data stores

## Workflow
1. Catalogue systems and data categories
2. Identify all cross-border flows
3. Map each flow to a transfer pathway
4. Output mapping report
5. Hand to data-protection-cn

## Output
Data flow mapping report.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
