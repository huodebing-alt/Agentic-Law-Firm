# Shareholder Resolution (PRC)

## Inputs
- Company name
- Matter
- Voting threshold (majority / 2/3 / unanimous)

## Workflow
1. Confirm threshold per 《公司法》 and articles
2. Draft resolution
3. Add signing block
4. Format per style guide
5. Output .docx

## Output
Shareholder resolution .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
