# Outbound Investment Roadmap

## Inputs
- Target country
- Target activity
- Investor entity
- Deal value

## Workflow
1. Map sensitive sector / sensitive country triggers
2. Build the filing sequence
3. Identify timing risk
4. Coordinate with M&A
5. Output roadmap

## Output
Outbound roadmap.

cases precisely.
