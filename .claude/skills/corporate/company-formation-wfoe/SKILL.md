# WFOE Formation

## Inputs
- Parent name
- Business scope
- Registered capital
- Location

## Workflow
1. Draft articles of association
2. Confirm scope against negative list
3. Build the registered-capital plan (subscription)
4. Map the SAMR + MOFCOM (record) flow
5. Output filing package

## Output
WFOE incorporation package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
