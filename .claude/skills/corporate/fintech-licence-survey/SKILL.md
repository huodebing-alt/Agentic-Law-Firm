# Fintech Licence Survey

## Inputs
- Activity description

## Workflow
1. Classify the activity
2. Map licences and the granting authority
3. Identify alternatives (white-label / partnership)
4. Output landscape memo
5. Hand to fintech-specialist for filing

## Output
Landscape memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
