# SAMR Merger Control Filing

## Inputs
- Parties' turnover (global, China, per party)
- Market overlap
- Deal value

## Workflow
1. Apply thresholds (RMB 4bn / 800m / 400m and per-party)
2. If triggered, pick simple vs normal case track
3. Draft notification (market definition + competitive assessment + remedies)
4. Coordinate with M&A timing
5. Output filing or no-filing memo

## Output
Filing package or no-filing memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
