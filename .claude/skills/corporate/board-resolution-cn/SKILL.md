# Board Resolution (PRC)

## Inputs
- Company name
- Matter (M&A / financing / related-party / officer change / ...)
- Date

## Workflow
1. Identify quorum and approval threshold
2. Draft the resolution
3. Add signing block
4. Format per style guide
5. Output .docx

## Output
Board resolution .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
