# Articles Amendment (PRC)

## Inputs
- Current articles
- Proposed changes

## Workflow
1. Redline the articles
2. Draft the supporting shareholder resolution
3. Map SAMR filing
4. Format per style guide
5. Output package

## Output
Amended articles + resolution + SAMR filing.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
