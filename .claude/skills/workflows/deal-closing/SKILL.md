# Deal Closing

## Inputs
- Deal type
- Closing date
- Source documents

## Workflow
1. Compile signing set
2. Draft closing certificates
3. Coordinate with opinions
4. Build closing bundle
5. Hand to hard-gate-keeper

## Output
Closing package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
