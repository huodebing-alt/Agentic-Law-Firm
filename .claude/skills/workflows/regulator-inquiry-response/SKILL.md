# Regulator Inquiry Response

## Inputs
- Inquiry text
- Subject matter

## Workflow
1. Triage and privilege scoping
2. Pull underlying facts
3. Draft response
4. Coordinate cross-functionally
5. Hand to hard-gate-keeper

## Output
Regulator response package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
