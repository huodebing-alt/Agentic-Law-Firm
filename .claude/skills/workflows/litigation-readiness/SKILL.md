# Litigation Readiness

## Inputs
- Matter summary

## Workflow
1. Triage cause-of-action and forum
2. Plan preservation
3. Draft pleadings
4. Build witness map and damages
5. Hand to hard-gate-keeper

## Output
Litigation readiness package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
