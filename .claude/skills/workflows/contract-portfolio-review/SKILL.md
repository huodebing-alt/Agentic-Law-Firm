# Contract Portfolio Review

## Inputs
- Portfolio directory or list of files

## Workflow
1. Extract text per file
2. Run type-specific checklists
3. Cluster issues across portfolio
4. Build a renegotiation priority list
5. Output portfolio memo

## Output
Portfolio memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
