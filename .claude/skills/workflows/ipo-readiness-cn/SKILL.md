# IPO Readiness (PRC)

## Inputs
- Listing venue
- Company info

## Workflow
1. Invoke task-decomposer
2. Dispatch to corporate / governance / IP / tax / labour / disputes
3. Aggregate gaps
4. Build remediation plan with timeline
5. Hand to hard-gate-keeper

## Output
IPO readiness report.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
