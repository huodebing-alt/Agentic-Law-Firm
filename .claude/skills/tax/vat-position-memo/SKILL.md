# PRC VAT Position Memo

## Inputs
- Transaction description
- Parties

## Workflow
1. Classify activity
2. Determine rate and special regime
3. Plan fapiao
4. Output memo
5. Hand to tax-vat-cn

## Output
VAT position memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
