# PRC CIT Position Memo

## Inputs
- Transaction description
- Parties
- Treaty (if any)

## Workflow
1. Determine residency and PE
2. Apply CIT Law + treaty
3. Document related-party position
4. Output memo
5. Hand to tax-corporate-cn

## Output
CIT position memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
