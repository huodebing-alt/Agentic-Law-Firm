# IIT Residency Memo

## Inputs
- Individual profile
- Travel history

## Workflow
1. Apply 183-day rule and habitual residence
2. Determine source of income
3. Apply treaty relief if cross-border
4. Output memo
5. Hand to individual-tax-cn

## Output
IIT residency memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
