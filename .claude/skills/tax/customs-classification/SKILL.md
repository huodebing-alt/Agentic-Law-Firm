# Customs Classification

## Inputs
- Product description
- Technical specs

## Workflow
1. Identify candidate HS codes
2. Apply General Rules of Interpretation
3. Recommend HS code
4. Output classification memo
5. Hand to tax-customs

## Output
Classification memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
