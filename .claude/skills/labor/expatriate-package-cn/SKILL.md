# Expatriate Package (Inbound)

## Inputs
- Home country
- Host city
- Role

## Workflow
1. Map residency and tax
2. Draft secondment agreement and housing terms
3. Coordinate with immigration
4. Format per style guide
5. Output package

## Output
Expatriate package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
