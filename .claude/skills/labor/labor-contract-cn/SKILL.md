# PRC Labour Contract

## Inputs
- Employee role
- Term
- Compensation

## Workflow
1. Pick template (fixed / open / project)
2. Fill role, location, compensation, working time
3. Add IP, confidentiality, non-compete (if applicable)
4. Format per style guide
5. Output contract

## Output
PRC labour contract .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
