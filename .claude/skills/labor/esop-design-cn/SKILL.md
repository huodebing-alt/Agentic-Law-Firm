# PRC ESOP Design

## Inputs
- Headcount
- Pool size
- Structure (LP / offshore)

## Workflow
1. Pick vehicle
2. Design vesting and leaver provisions
3. Draft plan and grant documents
4. Coordinate with tax
5. Format per style guide

## Output
ESOP plan package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
