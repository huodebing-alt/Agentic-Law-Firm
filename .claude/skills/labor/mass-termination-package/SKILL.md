# Mass Termination Package

## Inputs
- Headcount to be separated
- Reason

## Workflow
1. Confirm trigger and threshold
2. Plan timing and union consultation
3. Draft notice, severance schedule, filing
4. Hand to hard-gate-keeper
5. Output package

## Output
Mass termination package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
