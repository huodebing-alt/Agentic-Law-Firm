# Social Security Coordination

## Inputs
- Home country
- Host country
- Employee profile

## Workflow
1. Identify applicable treaty (e.g. CN-DE, CN-KR, CN-JP)
2. Apply certificate-of-coverage rules
3. Plan home-country continuation
4. Output coordination memo

## Output
Social security coordination memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
