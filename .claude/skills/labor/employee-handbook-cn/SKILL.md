# PRC Employee Handbook

## Inputs
- Industry
- Headcount
- Workplace

## Workflow
1. Draft sections (recruitment / contracts / compensation / leave / discipline / IP / confidentiality / departure)
2. Add complaint and consultation procedure
3. Coordinate with employee-representative procedures
4. Format per style guide
5. Output handbook

## Output
Employee handbook .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
