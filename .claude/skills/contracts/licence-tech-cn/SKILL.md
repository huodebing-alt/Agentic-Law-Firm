# Technology Licence (PRC)

## Inputs
- Licensed technology
- Scope
- Term
- Royalty

## Workflow
1. Pick template
2. Fill scope (territory, field, exclusivity)
3. Set royalty (fixed / running / milestones)
4. Add improvements and audit rights
5. Format per style guide

## Output
Tech licence agreement .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
