# Master Supply (PRC)

## Inputs
- Buyer
- Seller
- Products
- Term

## Workflow
1. Pick template
2. Fill quality, delivery, warranty, price-adjustment
3. Add tooling and IP terms if applicable
4. Format per style guide
5. Output .docx

## Output
Master supply agreement .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
