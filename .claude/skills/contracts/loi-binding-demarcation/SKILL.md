# LOI with Binding Demarcation

## Inputs
- Deal outline
- Exclusivity required (y/n)
- Costs allocation

## Workflow
1. Draft non-binding section (terms in principle)
2. Draft binding section (exclusivity, costs, confidentiality, governing law)
3. Label each clause clearly
4. Format per style guide
5. Output .docx

## Output
LOI .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
