# MSA + SOW Draft

## Inputs
- Services description
- Term
- Fees
- IP allocation

## Workflow
1. Draft MSA (general terms)
2. Draft SOW (specifics)
3. Allocate IP ownership
4. Add acceptance and SLAs
5. Format per style guide

## Output
MSA + SOW draft.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
