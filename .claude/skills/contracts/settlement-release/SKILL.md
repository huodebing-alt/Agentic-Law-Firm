# Settlement Release

## Inputs
- Dispute subject
- Payment terms
- Release scope

## Workflow
1. Confirm release scope and carve-outs
2. Draft payment terms
3. Add confidentiality and non-disparagement
4. Format per style guide
5. Hand to hard-gate-keeper

## Output
Settlement agreement .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
