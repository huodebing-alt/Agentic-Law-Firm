# Guarantee Contract (PRC)

## Inputs
- Guarantor
- Creditor
- Principal obligation
- Guarantee type

## Workflow
1. Distinguish 一般保证 vs 连带责任保证
2. Draft per Civil Code requirements
3. Add scope and term
4. Format per style guide
5. Output contract

## Output
Guarantee contract .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
