# IP Assignment (PRC)

## Inputs
- IP description
- Assignor
- Assignee
- Consideration

## Workflow
1. Confirm IP type and recordal requirement (CNIPA / MOFCOM)
2. Draft assignment with warranties
3. Prepare recordal documents
4. Format per style guide
5. Output package

## Output
IP assignment + recordal package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
