# Contract Redline

## Inputs
- Original .docx
- Change list or revised .docx

## Workflow
1. Load original
2. Apply tracked changes
3. Format per style guide
4. Output redline .docx
5. Append change log

## Output
Redlined .docx with change log.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
