# Trademark Licence

## Inputs
- Mark and class
- Scope
- Quality control
- Royalty

## Workflow
1. Pick template
2. Fill scope and quality-control covenants
3. Set royalty
4. Address recordal with CNIPA if applicable
5. Format per style guide

## Output
Trademark licence .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
