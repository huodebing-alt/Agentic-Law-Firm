# Franchise (PRC)

## Inputs
- Franchisor
- Franchisee
- Territory
- Fees

## Workflow
1. Confirm franchise-filing trigger (商务部 备案)
2. Draft the agreement
3. Add disclosure document
4. Format per style guide
5. Output package

## Output
Franchise agreement + disclosure document.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
