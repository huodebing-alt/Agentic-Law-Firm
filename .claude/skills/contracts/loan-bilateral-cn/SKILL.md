# Bilateral Loan (PRC)

## Inputs
- Lender
- Borrower
- Amount
- Tenor
- Security

## Workflow
1. Pick template
2. Fill amount, rate, tenor, repayment
3. Add security documents (mortgage / pledge / guarantee)
4. Coordinate with company-secretarial for corporate approvals
5. Format per style guide

## Output
Loan agreement and security suite .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
