# SaaS Subscription (PRC)

## Inputs
- Service description
- Term
- Fees
- Data location

## Workflow
1. Pick template
2. Fill subscription terms and SLAs
3. Add data location and processor terms (coordinate with PIPL)
4. Format per style guide
5. Output .docx

## Output
SaaS subscription agreement .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
