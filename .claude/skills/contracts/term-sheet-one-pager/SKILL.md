# Term Sheet One-Pager

## Inputs
- Deal economics
- Governance
- Exit

## Workflow
1. Draft Investor / Issuer / Valuation / Use of Proceeds / Governance / Exit / Closing Conditions
2. Demarcate binding clauses
3. Format per style guide
4. Output one-page .docx
5. Hand to deal-doc drafter

## Output
Term sheet (one page).

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
