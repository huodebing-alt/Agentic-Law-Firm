# One-Way NDA

## Inputs
- Disclosing party
- Receiving party
- Term
- Purpose

## Workflow
1. Pick one-way template
2. Fill term and purpose
3. Add return-or-destroy obligation
4. Format per style guide
5. Output .docx

## Output
One-way NDA .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
