# EPC / FIDIC Review

## Inputs
- Contract
- Party role (owner / contractor)

## Workflow
1. Identify FIDIC variant (Red / Yellow / Silver)
2. Run FIDIC-specific checklist
3. Flag deviations and missing protections
4. Output review memo
5. Hand to construction-contract

## Output
EPC review memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
