# Lease Review (PRC)

## Inputs
- Lease .docx
- Party role

## Workflow
1. Extract text
2. Run lease checklist (term, rent, escalation, fit-out, repair, default)
3. Flag risk and missing protections
4. Output review memo
5. Hand to real-estate-lease

## Output
Lease review memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
