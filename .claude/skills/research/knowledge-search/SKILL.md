# Knowledge Search

## Inputs
- Query
- Matter type

## Workflow
1. Query iManage / NetDocuments / Notion via the MCPs
2. Cluster results by topic
3. Surface the most relevant prior work
4. Output search report

## Output
Knowledge search report.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
