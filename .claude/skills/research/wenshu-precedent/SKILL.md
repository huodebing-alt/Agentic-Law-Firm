# Wenshu Precedent Search

## Inputs
- Topic
- Court level
- Year range

## Workflow
1. Query wenshu
2. Filter and rank
3. Summarise holdings
4. Output report

## Output
Precedent report.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
