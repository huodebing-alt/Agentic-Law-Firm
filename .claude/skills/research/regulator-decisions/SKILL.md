# Regulator Decisions

## Inputs
- Regulator (SAMR / CSRC / MOFCOM / NDRC / CAC / NMPA)
- Topic

## Workflow
1. Query the relevant regulator MCP
2. Filter by topic and year
3. Summarise positions
4. Hand to requesting specialist

## Output
Regulator citations + summary.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
