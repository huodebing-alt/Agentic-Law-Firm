# Asset Division (PRC)

## Inputs
- Asset inventory
- Spouses

## Workflow
1. Classify community vs separate
2. Value and allocate
3. Plan transfers and tax
4. Format per style guide
5. Output plan

## Output
Asset division plan .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
