# PRC Evidence Index

## Inputs
- Evidence inventory

## Workflow
1. Number and title each item
2. State purpose-of-proof per item
3. Format per court rules and style guide
4. Output index
5. Hand to drafter

## Output
证据清单 .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
