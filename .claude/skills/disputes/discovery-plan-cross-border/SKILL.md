# Cross-Border Discovery Plan

## Inputs
- Litigation forum
- PRC entities involved
- Document inventory

## Workflow
1. Map document custodians
2. Identify PRC data-export blockers (DSL Art. 36, PIPL Art. 41)
3. Plan the request channels and timing
4. Output plan
5. Hand to disputes-arbitration-international

## Output
Discovery plan memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
