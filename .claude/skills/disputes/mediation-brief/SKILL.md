# Mediation Brief

## Inputs
- Matter summary
- BATNA / WATNA

## Workflow
1. Draft brief (issues / interests / options / criteria)
2. Include offer ladder
3. Format per style guide
4. Output brief

## Output

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
