# Preservation Application (PRC)

## Inputs
- Counter-party
- Assets to preserve
- Underlying claim

## Workflow
1. Confirm urgency and security required
2. Draft 财产保全申请书
3. Compile evidence
4. Format per style guide
5. Hand to hard-gate-keeper

## Output
财产保全申请书 .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
