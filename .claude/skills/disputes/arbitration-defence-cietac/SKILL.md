# CIETAC Defence

## Inputs
- Request for arbitration
- Defence theory

## Workflow
1. Parse the request
2. Draft defence and counterclaim per CIETAC Rules
3. Compile evidence
4. Plan tribunal selection
5. Hand to hard-gate-keeper

## Output
CIETAC defence package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
