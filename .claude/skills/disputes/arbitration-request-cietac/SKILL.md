# CIETAC Request for Arbitration

## Inputs
- Arbitration clause
- Claims
- Damages basis

## Workflow
1. Confirm clause validity
2. Draft request per CIETAC Rules
3. Compile evidence
4. Plan tribunal selection
5. Hand to hard-gate-keeper

## Output
CIETAC request package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
