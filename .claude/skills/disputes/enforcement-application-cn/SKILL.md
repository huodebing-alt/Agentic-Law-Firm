# Enforcement Application (PRC)

## Inputs
- Judgment / award
- Asset findings

## Workflow
1. Confirm enforceability (no appeal / award final)
2. Identify assets
3. Draft 执行申请书
4. Compile evidence
5. Hand to hard-gate-keeper

## Output
执行申请书 .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
