# Antitrust Training

## Inputs
- Audience (sales / procurement)
- Industry

## Workflow
1. Outline 30 slides
2. Cover horizontal / vertical / abuse
3. Add do-and-don't list
4. Draft quiz
5. Output pack

## Output
Antitrust training pack.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
