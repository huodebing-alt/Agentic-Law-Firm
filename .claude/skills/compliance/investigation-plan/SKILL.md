# Investigation Plan

## Inputs
- Allegation summary
- Reporting channel
- Privilege regime

## Workflow
1. Triage scope and privilege
2. Issue preservation hold
3. Design interview list and document list
4. Set timeline and reporting cadence
5. Output plan

## Output
Investigation plan .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
