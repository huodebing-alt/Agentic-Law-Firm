# Data Handling Policy (PRC)

## Inputs
- Data inventory
- Industry

## Workflow
1. Classify data per DSL
2. Define lifecycle controls
3. Draft cross-border transfer rules
4. Add incident response
5. Format per style guide

## Output
Data handling policy .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
