# Interview Outline

## Inputs
- Interviewee role
- Allegation summary
- Documents collected

## Workflow
1. List Upjohn-style warnings (or PRC equivalent)
2. Draft chronological questions
3. Add follow-up branches
4. Add closing questions (who else, what documents)
5. Output outline

## Output
Interview outline .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
