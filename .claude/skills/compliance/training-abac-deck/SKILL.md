# ABAC Training

## Inputs
- Audience
- Industry
- Geographic footprint

## Workflow
1. Outline 30 slides (concepts / risks / scenarios / red flags / hotline)
2. Draft script
3. Draft 10-question quiz
4. Format per style guide
5. Output pack

## Output
ABAC training pack (deck + script + quiz).

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
