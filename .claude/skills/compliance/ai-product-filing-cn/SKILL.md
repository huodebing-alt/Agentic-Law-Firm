# AI Product Filing (PRC)

## Inputs
- Product description
- Audience (consumer / enterprise)
- Modalities

## Workflow
1. Map filings (algorithm filing / generative AI security assessment)
2. Audit training data for IP and personal-data compliance
3. Draft AI-incident response plan
4. Output filing package or no-filing memo

## Output
Filing package or no-filing memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
