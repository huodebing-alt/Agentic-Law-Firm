# Supplier Code of Conduct

## Inputs
- Industry
- Tier-1 supplier list

## Workflow
1. Pull ESG / ABAC / labour standards
2. Draft code with audit and remediation provisions
3. Format per style guide
4. Output code

## Output
Supplier code of conduct .docx.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
