# Compliance Program Design

## Inputs
- Industry
- Geographic footprint
- Current state

## Workflow
1. Gap-analyse vs ISO 37301 / GB/T 35770 / FCPA Resource Guide
2. Design policies, controls, training
3. Draft compliance manual outline
4. Set up monitoring cadence
5. Hand to compliance-program agent

## Output
Compliance program outline.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
