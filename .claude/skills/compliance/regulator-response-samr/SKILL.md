# Regulator Response (SAMR)

## Inputs
- Request text
- Underlying facts
- Privileged materials

## Workflow
1. Parse the request and the authority cited
2. Identify privileged vs producible materials
3. Draft response and supporting index
4. Hand to hard-gate-keeper
5. Output package

## Output
Regulator response package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
