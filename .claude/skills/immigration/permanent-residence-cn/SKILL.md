# PRC Permanent Residence

## Inputs
- Applicant profile
- Eligibility category

## Workflow
1. Confirm eligibility category
2. Compile documents
3. Draft application
4. Plan filing
5. Output package

## Output
Permanent residence package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
