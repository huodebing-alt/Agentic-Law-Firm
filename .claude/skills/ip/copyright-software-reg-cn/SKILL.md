# Software Copyright Registration (PRC)

## Inputs
- Software name
- Author
- Owner

## Workflow
1. Prepare title page and source-code sample
2. Fill application form
3. Prepare authorisation documents
4. Output package
5. Hand to copyright-cn

## Output
Software registration package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
