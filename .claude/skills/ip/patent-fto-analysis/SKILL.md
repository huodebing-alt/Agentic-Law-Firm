# FTO Analysis

## Inputs
- Product description
- Target jurisdiction

## Workflow
1. Identify search keywords and classifications
2. Query patent databases
3. Cluster relevant patents
4. Map relevance per claim
5. Output FTO memo

## Output
FTO memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
