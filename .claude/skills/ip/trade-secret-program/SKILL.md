# Trade Secret Program

## Inputs
- Company info
- Industry
- Crown jewels

## Workflow
1. Identify trade secrets
2. Draft marking and classification rules
3. Draft access controls
4. Draft training materials
5. Draft departure protocol

## Output
Trade secret program package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
