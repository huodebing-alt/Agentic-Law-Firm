# UDRP Complaint

## Inputs
- Domain
- Complainant rights
- Registrant info

## Workflow
1. Confirm complainant rights
2. Identify bad-faith indicators
3. Draft complaint per UDRP Rules
4. Compile annexes
5. Output complaint

## Output
UDRP complaint.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
