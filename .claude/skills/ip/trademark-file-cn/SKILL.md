# Trademark Filing (PRC)

## Inputs
- Mark
- Classes and goods/services
- Applicant

## Workflow
1. Pick classes and items
2. Prepare specimens
3. Draft application form
4. Add power of attorney
5. Output package

## Output
Application package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
