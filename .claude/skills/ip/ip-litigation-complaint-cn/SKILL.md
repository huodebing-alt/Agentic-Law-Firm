# PRC IP Complaint

## Inputs
- Asserted IP
- Accused product
- Damages basis

## Workflow
1. Confirm subject-matter and personal jurisdiction (IP court)
2. Plan evidence preservation
3. Draft complaint and damages calculation
4. Output complaint
5. Hand to hard-gate-keeper

## Output
Complaint and damages memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
