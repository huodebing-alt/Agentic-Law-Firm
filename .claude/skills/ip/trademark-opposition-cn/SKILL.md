# Trademark Opposition (PRC)

## Inputs
- Opposed application number
- Grounds
- Prior rights

## Workflow
1. Pull the opposed application
2. Identify grounds (Art. 30, Art. 13, etc.)
3. Compile prior-rights evidence
4. Draft opposition
5. Output package

## Output
Opposition package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
