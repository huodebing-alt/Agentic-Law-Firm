# Patent Drafting (PRC Invention)

## Inputs
- Invention disclosure
- Prior art search results

## Workflow
1. Identify novelty and inventive step
2. Draft claims (independent + dependent)
3. Draft specification and background
4. Draft abstract
5. Output filing pack

## Output
Patent application package.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
