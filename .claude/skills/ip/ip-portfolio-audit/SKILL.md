# IP Portfolio Audit

## Inputs
- Portfolio export (CSV / docket)

## Workflow
1. Categorise by asset type and territory
2. Identify lapse risk and renewals
3. Identify gaps in coverage
4. Recommend prune candidates
5. Output audit report

## Output
Portfolio audit report.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely.
