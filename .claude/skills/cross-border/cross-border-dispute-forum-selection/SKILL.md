# Cross Border Dispute Forum Selection

## Inputs
- Parties
- Dispute description

## Workflow
1. Apply each jurisdiction's jurisdiction-clause enforceability
2. Apply NY Convention recognition matrix
3. Output forum-selection memo

## Output
Forum-selection memo.

Every client-facing document follows `docs/DOCUMENT_STYLE_GUIDE.md`.
No emoji. No decorative colour. Black on white. Cite statutes and
cases precisely per jurisdiction convention.
